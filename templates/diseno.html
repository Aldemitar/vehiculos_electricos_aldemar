Mapa de Endpoints

El sistema expone los siguientes endpoints organizados por recurso:

  • GET /vehiculos_registro - Listar vehículos
  • GET /vehiculos/add - Formulario para agregar vehículo
  • POST /vehiculos/add - Agregar vehículo
  • POST /vehiculos/delete/{vehiculo_id} - Eliminar vehículo (form)
  • DELETE /vehiculos/{vehiculo_id} - Eliminar vehículo
  • GET /vehiculos/edit/{vehiculo_id} - Editar vehículo
  • POST /vehiculos/update/{vehiculo_id} - Actualizar vehículo
  • GET /vehiculos?marca=X - Filtrar vehículos por marca
  • GET /vehiculos_eliminados - Ver vehículos eliminados
  • POST /vehiculos/restaurar/{vehiculo_id} - Restaurar vehículo
  • GET /baterias - Listar baterías
  • GET /baterias/add - Formulario para agregar batería
  • POST /baterias/add - Agregar batería
  • POST /baterias/delete/{bateria_id} - Eliminar batería
  • GET /baterias/edit/{bateria_id} - Editar batería
  • POST /baterias/update/{bateria_id} - Actualizar batería
  • GET /baterias/eliminadas - Ver baterías eliminadas
  • POST /baterias/restaurar/{bateria_id} - Restaurar batería
  • POST /baterias/{bateria_id}/asociar - Asociar batería a vehículo
  • POST /baterias/{bateria_id}/desasociar - Desasociar batería
  • GET /estadisticas - Módulo estadístico
  • GET /planeacion - Ver planeación del proyecto
  • GET /diseno - Ver diseño del proyecto
