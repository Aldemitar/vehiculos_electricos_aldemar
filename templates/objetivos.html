Objetivo del Proyecto

Sistema de gestión de vehículos eléctricos y sus baterías, desarrollado con FastAPI, SUPABASE, SQLModel y PostgreSQL.

🎯 Objetivo General

El propósito de este proyecto es construir una plataforma digital intuitiva, eficiente y segura que permita llevar un control preciso de los vehículos eléctricos y sus respectivas baterías. Se busca facilitar tanto la gestión administrativa como técnica del inventario, brindando herramientas visuales e interactivas que simplifiquen el registro, monitoreo, análisis y mantenimiento de los recursos tecnológicos, apoyando así procesos sostenibles, tecnológicos e innovadores dentro del ámbito de la movilidad eléctrica.

🎯 Objetivos Específicos
  • 📌 Modelar adecuadamente la estructura de datos para representar entidades como vehículos y baterías, usando SQLModel y SQLAlchemy, estableciendo relaciones uno a uno y controlando estados como “eliminado” o “activo”.
  • 📌 Implementar un conjunto de vistas web con Jinja2 que permitan operaciones CRUD para ambas entidades, incluyendo formularios dinámicos, validaciones, y retroalimentación visual clara.
  • 📌 Habilitar la asociación y desasociación directa entre vehículos y baterías desde la interfaz gráfica, asegurando integridad de datos y usabilidad.
  • 📌 Incorporar almacenamiento externo de imágenes mediante Supabase, permitiendo la subida, gestión y visualización de fotografías para cada vehículo o batería.
  • 📌 Desarrollar un módulo de estadísticas visuales que brinde información útil para la toma de decisiones, como conteo por marca, porcentaje de baterías asignadas o en estado crítico.
  • 📌 Garantizar la disponibilidad del sistema mediante su despliegue en Render, asegurando acceso remoto desde cualquier dispositivo con conexión a internet.
  • 📌 Diseñar una interfaz responsiva y moderna, basada en Bootstrap, que permita a cualquier usuario navegar fácilmente desde desktop o móviles.

💡 Justificación

En el contexto actual de transición hacia tecnologías sostenibles, la movilidad eléctrica representa una necesidad urgente. Este proyecto responde a la creciente demanda de herramientas digitales que ayuden a organizar, controlar y visualizar información clave sobre recursos tecnológicos verdes. El sistema propuesto representa una contribución significativa al ecosistema de soluciones limpias, permitiendo a empresas o instituciones gestionar flotas de forma eficiente, reducir tiempos operativos y tomar decisiones informadas.

🧩 Alcance del Proyecto

El sistema cubre los aspectos más relevantes para una gestión operativa básica y funcional de vehículos eléctricos y sus baterías. Incluye módulos de registro, edición, eliminación (con recuperación), asignación de recursos, estadísticas gráficas y almacenamiento multimedia. No contempla en esta versión funcionalidades como geolocalización, alertas automáticas o mantenimiento predictivo, pero deja una base sólida para futuras extensiones.
